Exercise 1.3
============

> Can you think of other languages that use
> `--` for comments?

According to [rosettacode](http://rosettacode.org/wiki/Comments),
the following languages also use `--` for comments:

- Ada
- Haskell
- SQL
- Babel
- MAXScript
- Rexx, NetRexx, OoRexx
- Sather
- VHDL
- Euphoria

It does not metion Eiffel, which also uses `--` for comments.
